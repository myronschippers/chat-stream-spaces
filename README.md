# Chat Streams Spaces

A test of the GetStream React SDK. Building out a chat app clone of Google Spaces.

## Getting Started

Used Vite for the React with Typescript build

- `yarn vite`
    - select React
    - select Typescript
- Create `README.md`
- `git init`
- `git commit -m "initial commit"`
